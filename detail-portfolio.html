Live 33bot

live.33bot.io es la página donde en tiempo real puedes ver las sessones del bot y las conversaciones que la gente esta teniendo con dicho bot.

Pueden parar un bot para hablar con el usuario o conectarse como agente para que cuando el usuario necesite ayuda le salte un alerta sonora y visual para acceder al detalle del chat y hablar con el usuario.

Esta pensado para que pueda haber multiples agentes gestionando el mismo chat.
